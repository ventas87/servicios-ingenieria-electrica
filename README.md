## Despliegue

Este es un sitio estático desplegado en **Vercel**. Cualquier `git push` a la rama `main` despliega automáticamente.

## Estructura

```
.
├── index.html       # Página principal
├── vercel.json      # Configuración de Vercel
└── img/             # Imágenes del sitio (logo principal y logo SEC)
    ├── logo1.png
    └── logo2.png
```

## Agregar imágenes (logos)

Las imágenes binarias deben subirse directamente desde GitHub web:

1. Ve a la carpeta `img/` en este repo
2. Click en **Add file → Upload files**
3. Arrastra `logo1.png` (logo HS Ingeniería) y `logo2.png` (logo SEC)
4. Commit — Vercel redespliega automáticamente
